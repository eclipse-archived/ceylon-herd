Ceylon Herd is Free Software, licensed under the Affero GPL license, which means you are free to download its source code and use the Ceylon Herd code as you wish, as long as any publicly-deployed modified version of Ceylon Herd provides all users access to its source code at no charge. Ceylon Herd uses the following open-source technologies:
